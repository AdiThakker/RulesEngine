# RulesEngine

A dynamic rules engine which can be used to execute simple logic / decisions via configuration.

e.g. of a Rule Configuration
<Rule priority ="1">
        <RuleEvaluations>
          <RuleEvaluation propertyName ="" criteria ="" value ="" />
          <RuleEvaluation propertyName ="" criteria ="" value ="" />
        </RuleEvaluations>
        <RuleDecisions>
          <RuleDecision propertyName ="" value =""/>
        </RuleDecisions>
</Rule>
